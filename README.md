# 💚 Saúde Solidária

Sistema de pesquisa para um fundo colaborativo de saúde.

## 📋 Sobre o Projeto

O Saúde Solidária é uma plataforma de pesquisa para um fundo colaborativo onde funcionários contribuem com um percentual do salário e têm reembolso de medicamentos e exames.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript (Chart.js)
- **Backend**: Node.js + Express
- **Banco de Dados**: PostgreSQL (Render)
- **Hospedagem**: GitHub Pages (Frontend) + Render (Backend)

## 📊 Funcionalidades

- Formulário de pesquisa com campos para dados pessoais e familiares
- Painel administrativo com gráficos e estatísticas

- Limpeza do banco de dados (com confirmação)

## 🎨 Responsividade

O sistema é totalmente responsivo, funcionando em:
- Desktop (acima de 1024px)
- Tablet (768px a 1024px)
- Mobile (até 768px)
